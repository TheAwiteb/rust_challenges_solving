# Challenges.rs

Challenges I solve from [CodeWars](https://CodeWars.com) in [Rust](https://www.rust-lang.org/)

## Requirements
* [Rust](https://www.rust-lang.org/tools/install)

## Installation
```bash
$ git clone https://github.com/TheAwiteb/challenges.rs
$ cd challenges.rs
```

## Running
```bash
# Path is challenges.rs
$ cargo run
```
